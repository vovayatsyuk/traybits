# Traybits

Bits of information fetched with javascript in the system tray.

<img alt="Screenshot of the Traybits app" src="./media/screenshot-light.webp" width="840">

## Developement

```
# run watcher
npm run tauri dev

# update icon
npm run tauri icon media/app-icon.png

# build binaries
npm run tauri build
```
